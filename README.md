# Calculator App

## Frontend

1. Navigate to the `frontend` directory:

    ```bash
    cd frontend
    ```

2. Install dependencies:

    ```bash
    npm i dependencies
    ```

3. Run the development server:

    ```bash
    npm run start
    ```
    
## Backend

1. Navigate to the `backend` directory:

    ```bash
    cd backend
    ```

2. Install dependencies:

    ```bash
    npm i dependencies
    ```

3. Run the backend server:

    ```bash
    npm run start
    ```
